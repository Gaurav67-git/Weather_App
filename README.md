# Weather Forecast Web Application

## Description
A JavaScript weather application that shows real-time weather data for any city using a weather API.

## Features
- Search weather by city name
- Display temperature, humidity, and conditions
- Dynamic UI updates
- Error handling for invalid city

## Tech Stack
- HTML
- CSS
- JavaScript
- Weather API (e.g., OpenWeather)

## How to Run
1. Clone/download the repository
2. Add your API key in the JavaScript file
3. Open `index.html` in browser
